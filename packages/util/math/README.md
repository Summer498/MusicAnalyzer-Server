# @music-analyzer/math

Utility collection for mathematical operations used across MusicAnalyzer.

## Features
- Basic functions such as `mod`, power utilities and decimal rounding.
- Array and vector helpers (`getZeros`, `getRange`, `vSum`, `vSub`, `getOnehot`).
- Statistical reductions (`totalSum`, `median`, `argmax`, `argsMinMax`).
- Set operations (`isSubSet`, `isSuperSet`).
- Fast Fourier Transform utilities (`fft`, `Complex`, `correlation`).

These helpers are written in TypeScript and exported via `index.ts`.
